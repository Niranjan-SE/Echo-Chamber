# Echo-Chamber
Echo Chamber is a full-stack media bias analysis platform that automatically 
fetches articles from major Indian news outlets and uses Google Gemini AI to 
score how much of an echo chamber the discourse is. It detects opinion clusters, 
surfaces missing perspectives, measures toxicity, and lets the community vote on 
the health of each discussion. Built with React, Node.js, Express, PostgreSQL, 
and the Gemini API.
